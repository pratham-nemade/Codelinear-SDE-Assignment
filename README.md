# Codelinear SDE Assignment – Frontend Implementation

A pixel-perfect responsive frontend implementation based on the provided Figma design, built using React.js, Vite, and Tailwind CSS.

---

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* Lucide React

---

## Installation & Setup

Clone the repository:

```bash
https://github.com/pratham-nemade/Codelinear-SDE-Assignment
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Deployment

The project is deployed using Vercel.
